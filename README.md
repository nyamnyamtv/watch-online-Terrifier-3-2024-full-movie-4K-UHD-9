一些司法管辖区要求将受版权保护的作品以有形形式“固定”下来。版权通常会由多个作者共享，每个作者都有一套使用或授权该作品的权利，他们通常被称为权利持有者。这些权利通常包括复制、控制衍生作品、分发、公开表演以及署名等道德权利。
